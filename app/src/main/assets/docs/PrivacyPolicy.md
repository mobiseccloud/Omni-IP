# Omni-IP Privacy Policy

**Zero-Backend Policy**

Omni-IP utilizes a 'Zero-Backend Architecture'. This means:
* No user network traffic, PCAP files, or threat intelligence queries are hosted, harvested, transited, or stored on remote servers by Mobisec Cloud.
* All network packet filtering, sinkholing, and local PCAP generation are performed entirely on-device using the Android VpnService.
* Premium features utilizing third-party threat intelligence (AlienVault, AbuseIPDB, Shodan) communicate directly from the device using your own API keys.
